What is HTML?

HTML stands for HyperText Markup Language. It's the standard markup language for creating web pages.

Basic HTML Elements

  • Headings: h1, h2, h3, h4, h5, h6
  • Paragraphs: p
  • Links: a
  • Images: img
  • Lists: ul, ol, li
